What is WinDECT?

Welcome to the WinDECT project homepage. WinDECT is a high-performance, Windows-optimized Software Defined Radio (SDR) application designed specifically for the DECT 6.0 (1920-1930 MHz) frequency band.

Unlike Linux-based tools that require complex GNURadio installs, WinDECT is a native C++ application using the DirectShow/WinUSB stack for maximum throughput and minimum latency.

Core Features NEW!!
  • Full-Duplex Decoding: Listen to both FP (Fixed Part) and PP (Portable Part) streams simultaneously.
  • Packet Injection: Test base station security by crafting custom NWK-layer packets.
  • Real-time GMSK Demodulation: Optimized for Intel Core 2 Duo and higher.
  • DSC Audit Tool: Automatically identify unencrypted baby monitors and older cordless handsets.

Technical Details

DECT 6.0 uses 10 carrier frequencies in the US, with TDMA time slots (24 per frame). WinDECT handles the synchronization word (0xE98A) detection in hardware where supported, or via our SuperSync™ software algorithm.
